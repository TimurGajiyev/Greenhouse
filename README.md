# GreenHouse

Калькулятор подбора оборудования (equipment sizing) для гибридных и автономных
(off-grid) энергосистем: солнечные панели (PV), накопители энергии (BESS),
дизель-генераторы (DG). Референсная архитектура — NREL REopt.

Первый кейс — завод в Йемене (off-grid): проверка вендорского предложения
1500 kWp PV / 3132 kWh BESS / 1000 kW DG.

## Состояние

Версия v0.2 — шаги 1–3 из 9 закрыты:

1. Фундамент: окружение, git, скелет репозитория.

Дорожная карта шагов 4–9 — в `KICKOFF_PROMPT.md`, контекст проекта — в `CLAUDE.md`.

## Как запустить (Windows)

```powershell
python -m venv .venv                  # создать окружение (один раз)
.venv\Scripts\activate                # активировать (в каждом новом терминале)
pip install -r requirements.txt      # восстановить зависимости
pytest                               # прогнать тесты
```

## Упрощения v1 (сознательные)

- Топливо дизеля задано как $/kWh (без литров и удельного расхода).
- Нет деградации батарей, отказов оборудования, экспорта в сеть, CHP.
- Равномерный шаг времени (час/полчаса/сутки); месячные ряды — позже.
